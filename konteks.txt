==================================================
IDENTITAS PROJECT
==================================================

Nama project:
- battleground-calc

Database:
- battleground_calc

Environment / stack:
- Laragon
- CodeIgniter 4
- CodeIgniter Shield
- MariaDB/MySQL
- Bootstrap
- PHP 8.3+
- Dark mode only

Konsep aplikasi:
- aplikasi admin untuk manajemen score turnamen battleground / FF / PUBG style
- 1 tournament punya banyak pot
- 1 pot punya banyak team
- 1 team punya banyak score per game
- score dihitung dari rank + kill
- leaderboard per pot

Modul/controller utama yang sudah ada:
- Dashboard
- TournamentController
- PotController
- TeamController
- ScoreController
- LeaderboardController
- ImportController
- Auth via Shield

View dipisah per modul, contoh:
- app/Views/layouts/main.php
- app/Views/dashboard/index.php
- app/Views/scores/index.php
- app/Views/scores/_calculator.php
- app/Views/imports/registrations.php
- app/Views/imports/_preview.php
- dan view terkait lain

LOGIKA BISNIS YANG WAJIB DIPERTAHANKAN
==================================================

Status tournament:
- belum_mulai
- start
- selesai

Aturan:
- saat tournament selesai:
  - semua edit terkunci
  - score tidak bisa diubah
  - pot/team tidak bisa ditambah/edit/hapus
  - tampilan boleh read-only
- saat belum_mulai atau start:
  - score boleh diedit
  - pot/team boleh dikelola

Mapping placement point:
- 1 => 12
- 2 => 9
- 3 => 8
- 4 => 7
- 5 => 6
- 6 => 5
- 7 => 4
- 8 => 3
- 9 => 2
- 10 => 1
- 11 => 0
- 12 => 0
- selain itu => 0

Total point:
- total point = placement point + kill point

Leaderboard:
- total score = SUM(total_point)
- urut DESC
- tie-break sederhana yang aman
